# 局域网炸金花

一个给 NAS / Docker 部署的轻量炸金花网页游戏。手机和电脑都可以在同一个局域网内打开浏览器一起玩。

## 功能

- 房间号加入，同一个局域网内分享链接即可进房
- 2 到 6 人实时牌局
- 看牌、闷跟、明跟、加注、比牌、弃牌
- 自动洗牌、发牌、判型、结算、下一局
- 手机端适配，无需安装 App

## 本地运行

```bash
node server.js
```

打开 `http://localhost:3000`。

## Docker 部署到 NAS

在本目录执行：

```bash
docker compose up -d --build
```

然后让朋友在同一局域网访问：

```text
http://NAS的局域网IP:3000
```

例如：

```text
http://192.168.1.20:3000
```

## 可配置项

可以在 `docker-compose.yml` 里改这些环境变量：

- `PORT`: 容器监听端口，默认 `3000`
- `STARTING_CHIPS`: 每位玩家初始筹码，默认 `1000`
- `ANTE`: 底注，默认 `10`
- `MAX_PLAYERS`: 房间最大人数，默认 `6`

## 当前规则

牌型大小为：

```text
豹子 > 同花顺 > 同花 > 顺子 > 对子 > 单张
```

本版本暂不启用各地差异较大的特殊规则，例如 `235 吃豹子`。看牌后跟注和加注按双倍支付；比牌需要先看牌。
